# QR Code Generator

A simple Windows app to generate QR codes. Enter text or a URL, choose a save location, and create QR codes quickly and easily.  

---

## Features
- User-friendly graphical interface (GUI)
- Choose save location for generated QR codes
- Generates QR codes from text or URLs
- Optional desktop shortcut
- Works on Windows PCs without Python installed

---

## Installation

1. Download `QRCodeGeneratorInstaller_vX.XX.XX.exe` from the Releases page.  
2. Double-click the installer and follow the prompts.  
3. (Optional) Choose to create a desktop shortcut.  
4. Launch the app and start generating QR codes!

---

## Usage

1. Open the app.  
2. Enter text or a URL in the input box.  
3. Click **"Choose Save Location"** to select where your QR code will be saved.  
4. Click **"Generate QR Code"** to create the QR code.  
5. The QR code image will be saved as `qr_code.png` in the folder you selected.

---

## Requirements
- Windows 10 or 11
- No Python needed; installer includes everything
